**Structure:** a `skills/` folder of 13 separate skills ([[The 13 Skills]]), each a `SKILL.md`; a `research/laziness/` folder backing the anti-laziness rules; `.claude-plugin/` and `.github/copilot-instructions.md` for agent integration; asset-processing `scripts/*.mjs` (the only "JavaScript"); example images ("Floria"). **No git releases/tags** - versioning lives in `CHANGELOG.md` only. See [[Taste Skill Changelog]].

**Positioning:** not a component library, not a hosted generator - a **portable constraint layer** ([[Agent Skills Format|Agent Skills and SKILL.md Format]]). Site lists live example sites built with it: Floria (`floria-landing-page.vercel.app`), Collective OS (`collectiveos.vercel.app`). Explicit disclaimer: "Taste Skill has no official token, coin, or crypto project."

**Scope:** landing pages, portfolios, redesigns - see [[Scope and Context]].
