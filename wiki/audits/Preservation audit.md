# Preservation audit

Confidence tag: EVIDENCE-BASED.

## Compiled Truth

Run in the **redesign** workflow. Lists **every URL, nav label, form field, and anchor changed** - the list *should be empty* unless the operator explicitly approved a change. Guards the §11.F "never changes silently" set: URL structure / route slugs, primary nav labels, form field names/order, brand logo, legal/consent/cookie copy. Because *"SEO migration is the #1 redesign risk,"* this audit is what stops a redesign from quietly tanking rankings. See [[Modernization Modes]].
