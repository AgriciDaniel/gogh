# Start Here

Welcome to the **Gogh** - a persistent, source-cited knowledge base about [[Taste Skill (Project)|Taste Skill]], the anti-slop frontend framework for AI coding agents.

## If you want to understand why it works

Start with [[Constraint Beats Coaxing]] (the through-line), then follow the concept chain: [[AI Slop]] -> [[Taste as the Moat]] -> [[The Three Dials]] -> [[Encoded Design Principles]] -> [[Design Review as Infrastructure]] -> [[Image-First Pipeline]] -> [[Agent Skills Format|Agent Skills and SKILL.md Format]].

## How this brain is organized

- **concepts/** - the theory (with the honest counter-arguments preserved).
- **dials/** - the three tunable aesthetic axes.
- **rules/** - every canonical threshold and ban, straight from the skill.
- **playbooks/** - end-to-end workflows.
- **audits/** - the checks that gate shipping.
- **entities/**, **reception/**, **sources/**, **gaps/** - people/tools, how the world reacted, the evidence, and open questions.

## Ground rules for maintaining it

- Every rule cites the canonical `SKILL.md` or `tasteskill.dev`. Unverified claims are marked. See [[Source Manifest Guide]].
- Confidence tags: **EVIDENCE-BASED** (from the skill / official site), **PRACTITIONER** (community craft), **CONTESTED** (genuine disagreement), **FOLKLORE** (repeated, unverified).
- Refresh on any upstream release. See [[Hot]] for the watch list.
