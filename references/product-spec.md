# Gogh Product Spec

## Buyer

Developers, indie hackers, design engineers, and agencies who build frontends with AI coding agents (Claude Code, Cursor, Codex, Windsurf) and want the output to look intentional and brand-fit rather than templated AI slop.

## Domain

Composable frontend design skill stacks for AI coding agents, combining
taste-skill v2, make-interfaces-feel-better, Impeccable, Anthropic
frontend-design, ui-ux-pro-max, and Vercel web-design-guidelines into audited
stack recommendations, dial settings, conflict notes, and upstream watch.

## Core Workflows

- Install and stack selection: choose which of the six skills apply, order them,
  and document any known conflicts before the agent starts work.
- Greenfield build with the full stack: set the taste baseline, retrieve styles
  and palettes, execute micro-interactions, enforce contracts, and run audits.
- Audit-first redesign: inspect the current interface first, choose the
  modernization mode, preserve URLs and brand identity, then apply the stack.
- Upstream watch: import new captures, run registry diff, and flag rule,
  version, license, or detector changes.
- Stack advisor report from a project brief: recommend skills, dial settings,
  execution order, risks, and audit checkpoints.

## Deliverables

- Cheat sheet for the six-skill stack.
- Stack decision one-pager.
- Unified pre-flight mega-checklist.
- Quickstart for installing and operating Gogh.
- Stack advisor reports with recommended skills, sequencing, conflicts, and
  source-cited audit notes.

## Promise

Turn six frontend design skills into one persistent, source-cited operating
brain for choosing, stacking, auditing, and refreshing AI-agent frontend design
guidance.

## Non-Promises

- No invented rules, thresholds, or dial values. Every rule cites a canonical
  upstream capture or source-ledger entry, and unverified claims are marked as
  such
- No shipping past a failed audit or an unrun required pre-flight check
- On redesigns, no destruction of existing URLs, routes, or brand identity without an explicit modernization-mode decision
- No copying of a client's real API keys, private briefs, or unreleased assets into brain notes
- No mutation of customer systems, production apps, accounts, or publishing
  pipelines in V1. V1 is advisory and read-only.
